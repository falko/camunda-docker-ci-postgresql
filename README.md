camunda-docker-ci-mysql
=======================

MySQL docker image for Jenkins CI builds base on [camunda-docker-ci-base][].

# Additional Packages

  - mysql server

# MySQL User

  - `root` without password
  - `camunda` with password `camunda`

# Database

  - `process-engine`
